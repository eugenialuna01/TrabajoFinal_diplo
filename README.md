# ⚽ TorneoPro
Sistema web de gestión de torneos deportivos desarrollado con el stack MERN.
La aplicación permite administrar torneos, equipos, jugadores, partidos y tablas de posiciones mediante un sistema de roles y permisos. 🏆

# 🚀 Características principales

- ✅ Gestión completa de torneos deportivos
- ✅ Administración de equipos y jugadores
- ✅ Creación de fixtures y partidos
- ✅ Carga de resultados deportivos
- ✅ Tabla de posiciones automática
- ✅ Sistema de autenticación con JWT
- ✅ Roles y permisos dinámicos
- ✅ Dashboard administrativo
- ✅ Interfaz moderna y responsiva
- ✅ Integración con APIs deportivas externas

---

# 👥 Roles del sistema

## 👤 Usuario

Puede visualizar la información general del sistema.

### Permisos

- Ver torneos
- Ver equipos
- Ver jugadores
- Ver partidos
- Ver tabla de posiciones

---

## 🧑‍🏫 Entrenador

Puede administrar su equipo y sus jugadores.

### Permisos

- Ver torneos
- Crear y editar equipos
- Crear, editar y eliminar jugadores
- Ver partidos
- Ver tabla de posiciones

---

## 👑 Administrador

Posee control total del sistema.

### Permisos

- CRUD de torneos
- CRUD de equipos
- CRUD de jugadores
- CRUD de partidos
- Gestión de resultados
- Gestión de usuarios
- Administración de standings
